# Password Strength Checker

This is a simple Python project that checks the strength of a password and generates its hashed version using SHA-256.

## Features
- Checks if a password is **Weak**, **Medium**, or **Strong**
- Hashes the password for security

## How to use
1. Download `password_checker.py`
2. Run the script using Python:
   ```bash
   python password_checker.py
